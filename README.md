# NetoSearch
Ferramenta de prospecção de e-commerces que encontra lojas online por produto e localização. Gera Google Dorks automaticamente, detecta plataformas de loja, remove marketplaces e analisa domínios para identificar oportunidades comerciais de forma rápida.

Intelligent e-commerce discovery tool that finds online stores by product and location. Generates advanced Google dorks, detects e-commerce platforms, removes marketplaces and analyzes domains to identify high-intent commercial opportunities.

# 🔎 Buscador de Lojas + Dork Lab

Ferramenta de prospecção inteligente para encontrar lojas online através de buscas avançadas.

A plataforma combina **busca por produto**, **geração automática de Google Dorks** e **análise de domínios**, permitindo descobrir e-commerces ativos em segundos.

## 🚀 Funcionalidades

- 🔎 Busca de lojas por produto e localização
- 🧠 Geração automática de Google Dorks
- 🏪 Detecção de plataformas de e-commerce
- 📊 Score comercial dos domínios encontrados
- ⚡ Identificação de oportunidades de alta intenção
- 🧹 Remoção automática de marketplaces
- 🌐 Análise de múltiplas páginas do Google

## 🎯 Casos de uso

- Prospecção comercial
- Pesquisa de mercado
- Descoberta de e-commerces por nicho
- Análise de concorrentes
- OSINT para e-commerce

## 🛠 Tecnologias

- JavaScript
- Node.js
- Web scraping
- Google Dorks
- APIs de análise de domínio

## ⚠️ Aviso

Este projeto é destinado apenas para **pesquisa, prospecção comercial e fins educacionais**.
